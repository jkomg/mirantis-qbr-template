// TAM TOOLS · DOCKER

Two tools. Pick one.

Both run offline against data on your machine — nothing leaves your network, and neither tool phones home. They serve different jobs and read data differently; each card below says how.

QBR DECK BUILDER
Configurator → Deck →
Turns one account's data into an 18-slide Quarterly Business Review — exec summary, health, usage, incidents, wins, risks, roadmap, and renewal math — ready to present or export to PPTX.
HOW TO USE
Start in the Configurator: paste a Salesforce export or type values in by hand, toggle slides on/off, then open the Deck. Its Tweaks panel adjusts tone, length, and framing per meeting.
DATA & EPHEMERALITY
Your draft lives only in this browser's localStorage. Nothing is written to disk unless you explicitly click Download — close the tab without saving and it's gone.
SLA PERFORMANCE REPORT
Portfolio → Account detail →
Aggregates support-ticket history across every account into one portfolio view — severity lifecycle (what's SLA-bound vs. not), resolution time, recurring themes — then lets you click into any account for its full ticket history.
HOW TO USE
Open it directly, no setup step. Click any account row to drill in; click a column header to re-sort. Reload the page any time to re-read the latest data.
DATA & EPHEMERALITY
Reads accounts/*.json live, fresh, on every load — a read-only mount, nothing cached or written back by the app. It has no memory between visits; update a file and reload to see the change.
EXTERNAL REQUESTS
Zero from either tool. Fonts are self-hosted; nothing here phones home.
