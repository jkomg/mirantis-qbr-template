Mirantis QBR Template — 3 directions
Same mock customer (Vertex Logistics, Q3 FY26), same three slides, three visual systems. Pick a direction and we'll build out the full ~16-slide template with the tweak sliders wired up.
A · Executive Brief
AUBERGINE + BURGUNDY
Restrained, classic consulting deck. Deep aubergine title slides, white content slides, thin red ribbon details, dense tables. Best for board-level customer execs and formal renewal reviews.
B · Mission Control
SALES NAVY + CYAN
Data-room / Grafana feel. Dense KPI tiles, sparklines, mono numerics, traffic-light colors. Best for technical IT leadership, ops VPs, and internal account-health reviews.
C · Editorial
GRADIENT SWATCHES
Magazine-style storytelling. Bold gradient covers, oversized numerals, pull-quotes from customer sentiment, conversational tone. Best for relationship reviews and customer-success-driven QBRs.

UPSIDE
Version-controllable. Sales ops can template per-customer files. AI agents can write it.
TRADE-OFF
Higher learning curve for TAMs who haven't seen JSON before.
GOOD FOR
Bulk generation (board pack for top 20 accounts) and AI-assisted drafting.
What stays the same across A / B / C
Section order, mock data, tweak dial behavior. Only the visual system changes per direction. We can also mix-and-match — e.g. Editorial covers + Mission Control content slides.
